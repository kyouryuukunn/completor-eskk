completor-eskk
====================

[eskk.vim](https://github.com/vim-skk/eskk.vim) completion support for
[completor.vim](https://github.com/maralla/completor.vim).


Install
-------

[Install completor.vim](https://github.com/maralla/completor.vim#install) first.

For [vim-plug](https://github.com/junegunn/vim-plug)

```vim
Plug 'kyouryuukunn/completor-eskk'
```

For [pack](https://github.com/maralla/pack)

```bash
$ pack install kyouryuukunn/completor-eskk
```
